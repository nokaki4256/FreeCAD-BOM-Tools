# FreeCAD部品表作成マクロアルファ版

日本語ベースです。他言語の方は翻訳してください。
This is based on Japanese. If you use another language, please translate it.

・概要
FreeCADにて部品表を作成するのに既存のものだと
うまい具合に作成出来なかったので
ChatGPTとGeminiに最低限の指示を丸投げと
修正指示でなんとか形にしたものです。

組図内にあるパーツ、リンクで挿入したパーツ・アセンブリ、fastenersの数量を集計して
FreeCADのスプレッドシートとCSVファイルを出力するものです。

もし使う場合、

マクロはFreeCADのマクロファイルに入れるか、

地道にFreeCADでマクロを作成してコピペするなりしてください。

・使い方

Macros内の各マクロをFreeCADに取り込んで
部品表作成マクロを実行すれば勝手に部品表が作られます。

前準備として

パーツの名称は「パーツの名称」＋「-000」としてください。

これはFreeCADの環境によって変わるかも知れませんが、同一名称の部品（例えばパーツA）だと

リンクを連続で読み込んだら１個目はパーツA、２個目はパーツA000、３個目はパーツA001のようになると思われます。

これを事前に回避して組図作成中の数量表示のために下４桁に-000を付けています。

これなら取り込んだとしても-000,-001のように増加してくれます。

また部品表を作成したときはこの下４桁は削除されるようになってます。


組図上に部品があるのに部品表に反映されない場合、

サブアセンブリ部品表リスト作成（なんか変な名前)の

「部品があるのに弾かれた場合ここに例外処理を入れる」に反映されない部品名称の何桁かを入れたら回避できるようです。

これは必要に応じて追加してください。

スプレッドシートで部品表のテンプレートを作成してないので

生のスプレッドシートに羅列されてるだけですが運用に問題はないはず。

＊）注意点

組図作成中に部品名称の中にメーカー名を入れるのを忘れたので部品表にメーカー名を入れる場合手打ちになります。

これは流石にどっかで修正します。

恐らく「部品名称（メーカー名）」で最終的に運用することになりますがまだ先の予定。


作成された部品表は順不同でソートされてません。

これもどっかで修正しますがまだ先の予定。
